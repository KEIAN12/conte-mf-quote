# 見積書プラグイン 使い方ガイド（社員向け）

Claude の Cowork で、話しかけるだけで MoneyForward の見積書下書きが作れます。
セットアップは1回だけ。以降はいつでも `/mf-quote` で使えます。

---

## 🛠 インストール（10分 / 代表と一緒に画面共有しながら）

### 1. Cowork を開く
Claude アプリの左上メニューから「Cowork」に切り替える。

### 2. プラグインを追加する
1. 左下の **「プラグイン」** を開く
2. **「マーケットプレイスを追加」** を押す
3. 以下の URL をコピペして入力：

   ```
   https://github.com/KEIAN12/conte-mf-quote
   ```

4. 追加できたら、表示された `Conte mf quote` の **「インストール」** を押す

### 3. 初回だけ MF と連携する（OAuth）
新しいチャットを開いて、こう入力：

```
/mf-setup
```

対話が始まるので：
1. **MF Client Secret** の入力を求められる → 代表から渡された値を貼り付ける
2. 自動でブラウザが開く → **ご自身の MF アカウントでログイン**（代表のアカウントではなく**自分の**）
3. 「claude-agent がアクセスを求めています」→ **「許可」** を押す
4. ブラウザに「✅ 認証成功」が出たら、タブを閉じて Cowork に戻る
5. Cowork に「🎉 セットアップ完了」と出たら終わり

## 📝 使い方

### 新規作成（おまかせ）
Cowork で話しかけるだけ：

```
見積書を作りたい
```

または：

```
/mf-quote
```

ボタンで選択肢が出るので、**タップで選ぶだけ**：

1. **作り方を選ぶ** — 「おまかせ／骨組みだけ／複製」
2. **宛先を入力** — お客様の名前を入れる
3. **件名を入力** — 案件名を入れる
4. **金額の目安** — 「18万」のように大雑把でOK
5. **税別/税込** — ボタンで選ぶ
6. **プレビュー確認** — AIが組み立てた見積が見える
7. **「反映する」ボタン** を押すと MF に保存される
8. **MFのリンク** が表示されるので、クリックで MF 管理画面へ

### 過去の見積から複製
```
去年のフォーラム配信の見積を複製して新しく作りたい
```

### 項目を変えたい時
プレビュー表示後に、そのまま話しかければ調整できます：
- 「音響費を2万に変えて」
- 「進行管理費いらない」
- 「件名を変えて」

### PDF で欲しい時
MF に反映したあとに：
```
PDFでダウンロードして
```

これで `~/Downloads/` に保存されます。
**デフォルトではダウンロードされないので、必要な時だけ言ってください。**

## ⚠️ 注意点

| できること | できないこと |
|---|---|
| ✅ 見積書の下書きを作成 | ❌ 発行（メール送付）はできない |
| ✅ 過去見積の複製・編集 | ❌ 見積書本体の削除はできない（MF管理画面で手動削除） |
| ✅ PDF ダウンロード（指示時のみ） | — |

**発行（クライアントへのメール送信）は必ず MF 管理画面から行ってください。**誤送信防止のため、Claudeからは発行しない仕様です。

## ❓ よくある質問

### Q. 自分が作った見積が他の人から見える？
**はい**、株式会社CONTEの全メンバーから閲覧できます（MFの仕様）。ただし「誰が作成・編集したか」は記録されるので、履歴は追跡できます。

### Q. 見積書本体に自分の名前が印刷される？
**いいえ**。会社名「株式会社CONTE」のみ。担当者名は書かれません。

### Q. 間違えて下書きを作ってしまった
MF管理画面から該当の見積書を削除してください。

### Q. 「連携が切れた」と出る
```
/mf-setup
```
でもう一度セットアップしてください。

### Q. Client Secret がわからない・紛失した
代表に連絡してください。
