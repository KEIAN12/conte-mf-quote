# 見積テンプレート集（CONTE標準）

件名や業務内容から、どのテンプレートを使うかClaudeが判断するための参照。
金額は税抜。実際のペイロードに落とすときは、金額目標に合わせて配分する。

## 判定キーワード

| 件名に含まれる語 | テンプレート |
|---|---|
| 配信 / ライブ / ストリーミング / zoom / YouTube | → イベント配信 |
| CM / 広告 / スポット / サイネージ | → テレビCM・動画広告制作 |
| ブランディング / VI / CI / ロゴ設計 / ブランド構築 | → ブランディング |
| Web / サイト / LP / HP / ECサイト | → Web制作 |
| 動画編集 / 映像編集 / 企業VP | → 動画編集 |
| 撮影 / 写真 | → 撮影 |
| AI / 業務効率化 / 自動化 / コンサル | → AI活用コンサル |
| パンフレット / チラシ / ポスター / リーフレット | → グラフィック制作 |

判断に迷う場合はユーザーに確認する（「〇〇の案件ということでいいですか？」）。

## 配分ルール（共通）

1. **金額目標（税抜）** から各項目の比率で仮配分
2. **1,000円単位で丸める**。端数は「進行管理費」「ディレクション費」等のまとめ項目で吸収
3. **項目数は5〜8個** が見積としてバランス良い（多すぎると煩雑、少なすぎると根拠薄い）
4. **詳細（detail）を活用** して内訳の透明性を出す（「カメラ・配信用PC・キャプチャ・配線他」等）

## テンプレートにない案件

「その他」としてヒアリングベースで組む。工数×単価ベースの項目を中心に、以下のような汎用項目で：

- 企画・設計費
- 制作費（主業務）
- 素材費
- ディレクション費
- 修正対応費
- 進行管理費

金額目標を聞いた上で、業務内容に応じて配分。
